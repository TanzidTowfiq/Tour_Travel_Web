# Tours & Travel Website

## Run Backend Server
```bash
cd backend
npm install
npm run dev
```
Now Backend is running on http://localhost:4000

## Run Frontend Server
```bash
cd frontend
npm install
npm run dev
```
Now Frontend is running on http://localhost:3000
